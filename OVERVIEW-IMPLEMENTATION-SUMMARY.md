## üìà Metrics & Formulas

### Average Order Value (AOV)
```
AOV = Total Revenue √∑ Number of Orders

Example:
Revenue: $50,000
Orders: 400
AOV = $50,000 / 400 = $125
```

### Percentage Change
```
Change % = ((Current - Previous) / Previous) √ó 100

Example:
Current Revenue: $50,000
Previous Revenue: $42,000
Change = ((50,000 - 42,000) / 42,000) √ó 100 = +19.0%
```

### Low Stock Threshold
```
Low Stock: countInStock ‚â§ 10 AND countInStock > 0
```

### Out of Stock
```
Out of Stock: countInStock = 0
```
